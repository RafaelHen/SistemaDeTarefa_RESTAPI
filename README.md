## DbContext
#### Ele que irá conectar meu sistema C# com o bancos de dados. Ele é tipo um assistente que ajuda o programa a conversar com o banco de dados. Ele sabe como guardar informações e como encontrar coisas lá dentro. O DbContext também fica de olho nas mudanças que fazemos nas informações, para poder salvar essas mudanças no banco de dados mais tarde.

![image](https://github.com/RafaelHen/SistemaDeTarefa_RESTAPI/assets/48704686/50bb9133-4d5d-4204-b54b-11eafd7b435e)
